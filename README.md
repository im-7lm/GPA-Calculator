# 🎓 GPA Calculator

A modern GPA calculator built with React and TailwindCSS to help students track their academic performance easily.

---

## ✨ Features

- Add courses with name, credits, and grade  
- Real-time GPA calculation  
- CGPA calculation with previous semesters  
- Editable grading system (A → F)  
- Auto-save using localStorage  
- Clean dark UI and fully responsive design
